# Ho-09-AT-03 — The dataset mark in the pane

**Goal**

A pane row whose full path is a remembered dataset's mountpoint wears one quiet glyph. Core gains the exact-mountpoint test as a pure function with tests; `PaneModel` gathers the mountpoint set from cached facts at commit time; `PaneView` renders the mark.

**Context**

ho-09 Decision 6. The indicator rides facts the Field already remembers—no probe, no promise when facts are absent. `ZFSTopology` (`Sources/PalanaCore/Field/ZFSTopology.swift`) owns the topology questions (`datasetContaining`, `wholeDatasetSelection`)—the new test lives beside them. `PaneModel.commit(host:path:entries:)` (`Sources/Palana/PaneModel.swift:311`) is where a successful read lands—facts are reachable there via `engine.field.facts(for:)`, memory-only, no wire. Rows render in `PaneView` (`Sources/Palana/PaneView.swift`); secondary facts wear `Theme.inkFaint`.

**Required Changes**

1. **`ZFSTopology.mountpointSet(in datasets: [ZFSDataset]) -> Set<String>`** — the mounted datasets' mountpoints, absolute paths only (`legacy`/`none` excluded), normalized without a trailing slash (`/` itself stays `/`). Public, DocC, beside the other topology questions.

2. **Tests** in `Tests/PalanaCoreTests/ZFSTopologyTests.swift`, matching the suite's style: mounted absolute mountpoints in, unmounted excluded, `legacy`/`none` excluded, trailing-slash normalization, empty list answers empty set.

3. **`PaneModel`** — on a successful `commit`, read `engine.field.facts(for: host)?.zfsTopology?.value` and store `datasetMountpoints: Set<String>` on the model (empty when the host is local, facts are absent, or zfs is). `facts(for:)` is an actor call—hop accordingly, but never call `discover` here: memory answers or nothing does. A row is marked when its entry is a directory and the entry's full path (the pane's path joined with the entry name, no trailing slash) is in the set. Put the join+lookup in one small method the view calls—keep the truth in the model.

4. **`PaneView`** — the marked row's name column gains a trailing `◆` in `Theme.inkFaint`, small (the file-size/date weight, not the name's). Nothing else moves. The glyph choice is the hands session's to prune—commit the seam, keep the mark quiet.
