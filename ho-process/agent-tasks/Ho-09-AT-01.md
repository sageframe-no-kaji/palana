# Ho-09-AT-01 — FieldOutline in the core

**Goal**

Implement the field view's pure state in PalanaCore: a `FieldOutline` value that builds display lines from remembered facts, pure cursor/expand/collapse transitions, a relative-age formatter, and a `Field.allFacts()` snapshot method. Full unit battery, no wire, no UI.

**Context**

ho-09 builds the summonable topology overlay. ho-07's law governs: everything that can be wrong is a pure value in the core, the Surface renders and owns nothing. The overlay (Ho-09-AT-02, a separate task) will render `FieldOutline.lines` and delegate every keystroke to these transitions. Read `Sources/PalanaCore/Field/HostFacts.swift` first—`Dated<Value>`, `Reachability`, `HostCapability`, `ZFSDataset` are the whole fact vocabulary and this task adds no new facts. `Sources/PalanaCore/Surface/` already holds pure Surface machinery (`SequenceRecognizer`, `PaneIntent`)—match its style and DocC discipline.

**Files**

- Create: `Sources/PalanaCore/Surface/FieldOutline.swift`
- Modify: `Sources/PalanaCore/Field/Field.swift` (add `allFacts()`)
- Create: `Tests/PalanaCoreTests/FieldOutlineTests.swift`

**Required Changes**

1. **`FieldOutline`** in `Sources/PalanaCore/Surface/FieldOutline.swift`. A `public struct FieldOutline: Equatable, Sendable`:

   - `public init(hosts: [String], facts: [String: HostFacts], localHost: String)` — `hosts` arrives already ordered (local first, then config order, exactly as `PalanaSession.hosts` holds it). `localHost` is `PalanaCore.localHostName`'s value passed in.
   - `public enum Line: Equatable, Sendable` with two cases carrying payload structs:
     - `case host(HostLine)` — `public struct HostLine`: `alias: String`, `isLocal: Bool`, `visited: Bool` (facts exist for the alias), `reachability: Reachability?`, `rememberedAt: Date?` (reachability's `discoveredAt`—every `discover` stamps it), `flavor: UserlandFlavor?`, `hasZFS: Bool`, `hasRsync: Bool` (from `capability.value.zfs`/`.rsync` non-nil), `expanded: Bool`, `datasetCount: Int` (remembered datasets, 0 when none).
     - `case dataset(DatasetLine)` — `public struct DatasetLine`: `host: String`, `name: String`, `mountpoint: String`, `pointable: Bool` (true only when `mounted == true` and `mountpoint` begins with `/`).
   - `public private(set) var lines: [Line]` (or a computed property—your call, but `Equatable` must hold and rebuilding must preserve cursor/expansion). Dataset lines appear under their host only while that host is expanded, in the order `zfsTopology` remembers them. The local host renders as a host line with `isLocal: true` and never carries facts.
   - `public private(set) var cursor: Int` — index into `lines`, clamped. Transitions, all `public mutating func`: `cursorDown()`, `cursorUp()`, `expand()` (on a host line with `datasetCount > 0`, marks expanded), `collapse()` (on a dataset line, collapses its host and moves cursor to that host line; on an expanded host line, collapses it; otherwise no-op).
   - `public struct Pointing: Equatable, Sendable { public let host: String; public let path: String }` and `public func pointing() -> Pointing?` — host line yields `(alias, "~")` (local yields `(localHost, "~")`), pointable dataset line yields `(host, mountpoint)`, non-pointable dataset yields nil.
   - `public func hostUnderCursor() -> String?` — the alias of the host line at the cursor, or the owning host of a dataset line. Nil on an empty outline.
   - `public mutating func update(facts: [String: HostFacts])` — rebuilds lines from new facts, preserving expansion set and keeping the cursor on the same line where it survives (same host alias, or same host+dataset name), clamping otherwise.
   - `public enum FieldAge` with `public static func describe(_ date: Date, now: Date) -> String` — "just now" under 60 seconds, then "Nm ago", "Nh ago", "Nd ago", integer truncation, never negative (a future date reads "just now"). Lives in the same file.

2. **`Field.allFacts()`** in `Sources/PalanaCore/Field/Field.swift`: `public func allFacts() -> [String: HostFacts]` — returns the in-memory snapshot, never touches the wire or the Conduit. One line plus DocC. Place beside `facts(for:)`.

3. **Unit battery** in `Tests/PalanaCoreTests/FieldOutlineTests.swift`, swift-testing (`@Suite`/`@Test`), matching the style of `Tests/PalanaCoreTests/FieldTests.swift`. Cover at minimum:

   - Line building: visited host with full facts, never-visited host, local host, unreachable host (reachability `.unreachable` renders with its `rememberedAt`).
   - Expansion: expand shows datasets in remembered order, collapse from a dataset line lands the cursor on the host, `legacy`/`none`/unmounted datasets appear with `pointable == false`.
   - Cursor: clamping at both ends, down/up across host and dataset lines.
   - Pointing: host → `~`, mounted dataset → mountpoint, unmounted dataset → nil, local → `(localHost, "~")`.
   - `update(facts:)`: a host remembered as BSD updates to BusyBox flavor in place (zencat's stale-flavor shape—build the before/after `HostFacts` by hand), expansion and cursor survive, cursor clamps when its dataset line vanishes.
   - `FieldAge.describe`: the four buckets plus the future-date guard.
   - `Field.allFacts()`: returns what discovery recorded, and a `Field` constructed with an empty cache answers `[:]`—add to `FieldTests.swift` or the new suite, your call.
